Steal SAM database hashes and other local hashes

The SAM database stores several hashes which are retrivable with the correct privileges. There are numerous techniques which can be used to get these hashes.

Registry hives

There are 3 relevant registry hives that can be used to extract local or domain hashes and credentials.(The SECURITY isn't necessary, but could be useful.)

- HKLM\SAM

- HKLM\SECURITY

- HKLM\SYSTEM

HKLM\SAM

This contains the local user accounts with their password hashes.

HKLM\SECURITY

Contains relevant information for Local Security Authority (LSA). E.g.: Chached domain credentials (DCC2), plaintext password

HKLM\SYSTEM

Stores the boot key which is used to encrypt the SAM database.

1st method

reg.exe

Run the cmd.exe with admin privilages (this is required for the reg.exe)

Create a copy of the files mentioned above with the reg.exe:

reg.exe save hklm\sam C:\User\user\tmp\SAM.save

reg.exe save hklm\system C:\User\user\tmp\SYSEM.save

reg.exe save hklm\security C:\User\user\tmp\SECURITY.save

These copies can be used to get the HASH from them.

2nd method

Shadow Copy

Create a shadow copy from the C:\ volume.

vmic shadowcopy call create Volume='C:\'

The output of the command contains the ShadowID which can be used to identify our created shadowcopy.

List all of the shadow copy and identify our by look for the same Shadow Copy ID

vssadmin list shadows

Get the path of our shadow copy and create a copy from the SAM, SECURITY and SYSTEM files from the copy.

copy [SHADOW_COPY_VOLUME_PATH]\windows\system32\config\SAM C:\Users\user\tmp\SAM.save

copy [SHADOW_COPY_VOLUME_PATH]\windows\system32\config\SECURITY C:\Users\user\tmp\SECURITY.save

copy [SHADOW_COPY_VOLUME_PATH]\windows\system32\config\SYSTEM C:\Users\user\tmp\SYSTEM<.save/p>

These copies can be used to get the HASH from them.

Get the hash

Automated

Impacket's secretsdump is a useful tool to obtain the hashes from the hive files.

It will print the boot key, any discovered credentials, and the hashes to standard output when using the default configuration, along with the hash type and its scheme./p>

python3 secretsdump.py -sam SAM.save -security SECURITY.save -system SYSTEM.save LOCAL

Manual

Crack the hash

There are several online and offline tool that can be used to crack the hash.

My favorite one is the Hashcat.

The Hashcat is well documented and provides several examples to use it correctly.
